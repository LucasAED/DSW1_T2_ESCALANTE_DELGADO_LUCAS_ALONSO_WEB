## 📋 Descripción del Proyecto

Aplicación web desarrollada con **HTML5, CSS3 y JavaScript (Vanilla)** para consumir los servicios de la API de Biblioteca.

El proyecto permite gestionar el inventario de libros y realizar préstamos en tiempo real, conectándose mediante **Fetch API** al Backend previamente desarrollado, permitiendo visualizar, registrar y eliminar datos de forma dinámica.

## 🛠️ Tecnologías Utilizadas

* **HTML5** (Estructura semántica)
* **CSS3** (Estilos y diseño responsivo)
* **JavaScript ES6+** (Lógica del cliente)
* **Fetch API** (Consumo de servicios REST)
* **DOM Manipulation** (Renderizado dinámico de tablas)

## 🚀 Instrucciones de Instalación y Ejecución

Siga estos pasos para utilizar el cliente web correctamente:

### 1. Requisito Previo (Backend)

Para que esta web funcione, el servicio Backend debe estar ejecutándose.

Asegúrese de que la API del repositorio Backend esté corriendo en Visual Studio.

### 2. Ejecutar la Web

1. Descargue este repositorio o ubique los archivos en su equipo (`index.html`, `styles.css`, `app.js`).
2. Haga doble clic en el archivo **`index.html`**.
3. La aplicación se abrirá en su navegador predeterminado.

*Nota: Al ser una conexión local, asegúrese de haber aceptado los certificados de seguridad de localhost en el navegador si es necesario.*
